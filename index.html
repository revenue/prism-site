Docs — 이 시스템은 무엇이고 어떻게 쓰는가

발표자료용 그래픽을 매일 자동 수집하고, 세그먼트로 호출하고, 값만 넣어 결과물을 출력하는 로컬 우선 시스템의 설명·구성·정책·가이드.

1 · 설명

세계 최고 소스 20곳(6 카테고리)에서 차트·인포그래픽·아이콘·일러스트·프레젠테이션 템플릿을 수집한다. 아이콘·일러스트는 실제 SVG까지 내려받고, 갤러리형은 메타(제목·썸네일·링크)만 인덱싱한다.

2 · 구성

  • 라이브러리 — 카테고리·타입·소스·검색으로 세그먼트 호출, 로고/썸네일 표시, 원본·다운로드.
  • 제너레이터 — 9종 템플릿 + 프리셋 + 스타일 설정 → SVG·PNG.
  • 에셋 스튜디오 — 아이콘 추출·리컬러·리사이즈, 이미지 팔레트 추출.
  • 요청내역 — 코멘트 요청과 해결 현황.

3 · 정책

  • 수집은 공개 RSS·오픈 API 우선(합법·안정). 갤러리형은 메타만.
  • 맥북 전원과 무관하게 GitHub Actions 클라우드에서 매일 11:00 KST 실행.
  • 라이선스는 아이템별 license 필드에 기록 — 재사용 전 확인.

4 · 가이드

  1. 테마: 우상단 버튼으로 라이트/다크 전환.
  2. 그래픽: 제너레이터 → 요소·프리셋 선택 → 값 입력 → 저장.
  3. 코멘트: 우하단 버튼 → 화면 클릭 핀 → 내용 입력 → "적용 요청" → 요청내역에서 처리 추적.
